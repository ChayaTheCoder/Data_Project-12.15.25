# Data_Project‑12.15.25

### Overview
This project is a **C‑based data analysis tool** that generates 1,000 random floating‑point values and allows users to explore key statistical operations through a menu‑driven interface.

### Features
- Calculates **minimum**, **maximum**, **average**, **median**, **mode**, **range**, **variance**, and **standard deviation**
- Finds the **index** and **occurrence count** of specific values
- Prints the first 100 elements of the dataset
- Sorts the array using a simple **bubble sort** algorithm
- Uses `<math.h>` for precision and `<time.h>` for randomization

### Compilation
```bash
gcc Data_Project.c -o Data_Project -lm
